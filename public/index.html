Buttondown and Netlify Functions Demo

This is an example of a custom HTML subscription form that uses a Netlify Function and the Buttondown API to add a subscriber to an email newsletter.

This is not a working example. Clone this repository and deploy it to Netlify to try it out yourself.

Deploy your own

A great place to start is to clone and deploy your own copy of this example. You can do that by clicking the button below.

Clicking the button will clone this repo and deploy it as a new site on Netlify
